Im Backend wurde nodejs verwendet, um Abfragen aufzugreifen.
npm install express
npm install cors
npm install fs

express stellt das Herzstück unseres Server da. Mit const app = express(); initialsieren wir die Express-App. Routing, Middleware und Verwalten von Anfragen findet darüber statt.
Über das app-Objekt greifen wir auf die Express-Anwendungen zu.
